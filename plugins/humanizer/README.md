# humanizer

> Remove AI writing patterns — strips corporate filler, hedging, and statistical language

## Install

```
/plugin install humanizer@wrench-plugins
```

Or add to your project `.claude/settings.json`:

```json
{
  "extraKnownMarketplaces": [
    { "name": "wrench-plugins", "source": { "source": "github", "repo": "WrenchAI/wrench-plugins" } }
  ]
}
```

Then: `/plugin install humanizer@wrench-plugins`
