# wrench-outreach

> Write B2B outreach sequences — email and LinkedIn, multi-step, persona-specific

## Install

```
/plugin install wrench-outreach@wrench-plugins
```

Or add to your project `.claude/settings.json`:

```json
{
  "extraKnownMarketplaces": [
    { "name": "wrench-plugins", "source": { "source": "github", "repo": "WrenchAI/wrench-plugins" } }
  ]
}
```

Then: `/plugin install wrench-outreach@wrench-plugins`
