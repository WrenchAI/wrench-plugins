# wrench-de-review

> Data engineering PR review — idempotency, dedup, cardinality assertions, schema contracts

## Install

```
/plugin install wrench-de-review@wrench-plugins
```

Or add to your project `.claude/settings.json`:

```json
{
  "extraKnownMarketplaces": [
    { "name": "wrench-plugins", "source": { "source": "github", "repo": "WrenchAI/wrench-plugins" } }
  ]
}
```

Then: `/plugin install wrench-de-review@wrench-plugins`
