# wrench-charts

> Brand-compliant Recharts chart components — type guide, tokens, tooltip patterns

## Install

```
/plugin install wrench-charts@wrench-plugins
```

Or add to your project `.claude/settings.json`:

```json
{
  "extraKnownMarketplaces": [
    { "name": "wrench-plugins", "source": { "source": "github", "repo": "WrenchAI/wrench-plugins" } }
  ]
}
```

Then: `/plugin install wrench-charts@wrench-plugins`
