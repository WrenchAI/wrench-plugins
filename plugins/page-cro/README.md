# page-cro

> Conversion rate optimization for marketing pages — hero, CTA, social proof

## Install

```
/plugin install page-cro@wrench-plugins
```

Or add to your project `.claude/settings.json`:

```json
{
  "extraKnownMarketplaces": [
    { "name": "wrench-plugins", "source": { "source": "github", "repo": "WrenchAI/wrench-plugins" } }
  ]
}
```

Then: `/plugin install page-cro@wrench-plugins`
