# seo-audit

> Comprehensive SEO audit — technical, on-page, Core Web Vitals, indexing

## Install

```
/plugin install seo-audit@wrench-plugins
```

Or add to your project `.claude/settings.json`:

```json
{
  "extraKnownMarketplaces": [
    { "name": "wrench-plugins", "source": { "source": "github", "repo": "WrenchAI/wrench-plugins" } }
  ]
}
```

Then: `/plugin install seo-audit@wrench-plugins`
