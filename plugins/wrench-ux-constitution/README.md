# wrench-ux-constitution

> UX governance — 11 principles, Six Pillars, Seven Empirical Gaps, pre-ship checklist

## Install

```
/plugin install wrench-ux-constitution@wrench-plugins
```

Or add to your project `.claude/settings.json`:

```json
{
  "extraKnownMarketplaces": [
    { "name": "wrench-plugins", "source": { "source": "github", "repo": "WrenchAI/wrench-plugins" } }
  ]
}
```

Then: `/plugin install wrench-ux-constitution@wrench-plugins`
