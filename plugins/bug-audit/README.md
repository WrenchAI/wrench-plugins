# bug-audit

> Structured audit for functional bugs, UX bugs, test coverage, and customer-surfaced issues

## Install

```
/plugin install bug-audit@wrench-plugins
```

Or add to your project `.claude/settings.json`:

```json
{
  "extraKnownMarketplaces": [
    { "name": "wrench-plugins", "source": { "source": "github", "repo": "WrenchAI/wrench-plugins" } }
  ]
}
```

Then: `/plugin install bug-audit@wrench-plugins`
